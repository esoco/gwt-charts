49  	 * How multiple data selections are rolled up into tooltips:
50  	 * <ul>
51  	 * <li>'category': Group selected data by x-value.</li>
52  	 * <li>'series': Group selected data by series.</li>
53  	 * <li>'auto': Group selected data by x-value if all selections have the same x-value, and by series otherwise.</li>
54  	 * <li>'none': Show only one tooltip per selection.</li>
55  	 * </ul>
56  	 * aggregationTarget will often be used in tandem with selectionMode and tooltip.trigger, e.g.:
57  	 * 
58  	 * <pre>
59  	 * // Allow multiple simultaneous selections.
60  	 * options.setSelectionMode(SelectionMode.MULTIPLE);
61  	 * // Trigger tooltips on selections.
62  	 * Tooltip tooltip = Tooltip.create();
63  	 * tooltip.setTrigger(TooltipTrigger.SELECTION);
64  	 * options.setTooltip(tooltip);
65  	 * // Group selections by x-value.
66  	 * options.setAggregationTarget(AggregationTarget.CATEGORY);
67  	 * 
68  	 * </pre>
69  	 * 
70  	 * @param aggregationTarget
71  	 */
