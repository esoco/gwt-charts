46  	 * How multiple data selections are rolled up into tooltips:
47  	 * <ul>
48  	 * <li>'category': Group selected data by x-value.</li>
49  	 * <li>'series': Group selected data by series.</li>
50  	 * <li>'auto': Group selected data by x-value if all selections have the same x-value, and by series otherwise.</li>
51  	 * <li>'none': Show only one tooltip per selection.</li>
52  	 * </ul>
53  	 * aggregationTarget will often be used in tandem with selectionMode and tooltip.trigger, e.g.:
54  	 * 
55  	 * <pre>
56  	 * // Allow multiple simultaneous selections.
57  	 * options.setSelectionMode(SelectionMode.MULTIPLE);
58  	 * // Trigger tooltips on selections.
59  	 * Tooltip tooltip = Tooltip.create();
60  	 * tooltip.setTrigger(TooltipTrigger.SELECTION);
61  	 * options.setTooltip(tooltip);
62  	 * // Group selections by x-value.
63  	 * options.setAggregationTarget(AggregationTarget.CATEGORY);
64  	 * 
65  	 * </pre>
66  	 * 
67  	 * @param aggregationTarget
68  	 */
